Симулятор квантового розподілення ключів (протокол BB84), розроблений для проекту по кібербезпеці в МАНі, 2021-2022 роки

Розробник - Єримович Андрій

Використання:

1. Запустіть файл Main.exe
2. Дотримуйтесь вказівок на екрані, щоб розпочати процес симуляції
3. Дочекайтесь виведення результатів симуляції
4. Для того щоб запустити симуляцію ще раз, натисніть 1

Примітка: У файла Main.exe немає зареєстрованого підпису розробника, тому антивірус Windows Defender, та інші, можуть розпізнавати його як вірусну програму. 
Проігноруйте попередження або добавте виключення для цього файлу - він безпечний.


Для доступу до коду відкрийте файл Main.py, у будь-якому редакторі коду.
